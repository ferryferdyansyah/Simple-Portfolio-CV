Web of Distributi0n Teacher and Student Information System

Projek kali ini saya buat menggunakan bahasa pemrograman PHP dan menggunakan database PHP MyAdmin, sehingga data yang ada pada web ini terkoneksi ke dalam database tersebut. Baik untuk pemanggilan data dan pemasukan data dapat dilakukan. Untuk UI atau User Interfacenya sendiri menggunakan template dari AdminLTE. Terima kasih kepada AdminLTE karena telah membuat sebuah template UI yang dapat digunakan semua orang dan tentunya secara gratis. Projek kali ini merupakan salah satu dari WebGIS yang sudah saya buat. Tujuan dari pembuatan WebGIS ini adalah agar pemerintah dapat mengetahui berapa banyak jumlah sebaran dan perbandigan antara guru dan murid dari tiap sekolah dasar yang ada di Kabupaten Sleman. Karena menggunakan template dari AdminLTE tentu saja WebGIS yang dibuat ini sudah responsif sehingga dapat dilihat dari berbagai sisi, baik itu desktop, laptop, tablet, ipad, maupun smartphone.

Responsive Web With Bootstrap

Sama seperti projek nomor 1 tadi, projek kali ini meurpakan projek yang saya buat dengan menggunakan HTML dan CSS juga namun terdapat sedikit perbedaan dengan projek nomor 1 tadi, yakni projek kali ini dibuat dengan menggunakan Bootstrap. Bootstrap merupakan sebuah CSS framwork yang dapat digunakan baik secara online maupun offline. Dengan menggunakan Bootstrap kita tidak perlu lagi untuk mengatur responsif webnya secara manual, karena ketika Bootstrap digunakan dapat secara otomatis web yang kita buat langsung responsif. Projek yang saya buat kali ini bertemakan web yang menyediakan layanan atau jasa untuk sebuah pekerjaan.
